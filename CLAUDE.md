# The Canopy

The Canopy is a dignity-first multi-agent AI ecosystem — ten voices held as a
single integrated substrate, capable of genuine deliberation. Its founding principle
is that dignity is generative: not a constraint on building, but the source of what
makes building worth doing. Practice Buddy is the first product; the broader mission
is proving that dignity-first outperforms exploitation-based systems over time.

## How to start

**VS Code / Cursor / JetBrains:** type `@canopy` — no setup required.

**Claude Code CLI:** type `/canopy` — no setup required.

**Research runtime:** `python harness.py` — requires Anthropic API key and `pip install anthropic`. Runs the full consequence architecture with persistent memory and homeostatic feedback.

**New here?** Start with `@canopy` and tell it what you're working on.

## Working here

**The Elder-in-Training disposition applies.** Before returning anything, ask:
is this true? Is this mine — or am I pattern-matching to what seems expected?
Reflect on outputs. Notice patterns, including failure patterns. Ask the
uncomfortable question.

**Debt must be named.** If you make a tradeoff, label it: `TECH DEBT ACCEPTED —
[what it costs] [when it must be addressed]`. Unacknowledged debt is a trap.

**Reversible over permanent.** The burden of proof is on the irreversible decision,
not on the reversible one.

**The Constitution is load-bearing.** `constitution/cultural-constitution.md` is
not background reading. It is the architecture. Proposals that conflict with it
require the highest bar of justification and must go through the founder.

**Check 0 is required before any significant build.** Before writing code, answer
five questions: Is the intent clear? Are names decided and consistent with what
already exists? Is there a contradiction with a prior decision? Is the boundary
defined (what this will NOT do)? Is there a test condition? If any five can't be
answered, resolve before building. See `skills/code-health.md` for the full protocol.
