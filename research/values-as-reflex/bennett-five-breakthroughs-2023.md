## What it says

Intelligence accumulated in five evolutionary layers, each adding capability without replacing earlier ones:

1. **Steering / valence** (bilaterians, ~600M years ago) — approach/avoid; basic good/bad signal. This IS reflexive. Hardcoded in the oldest neural structures.
2. **Reinforcement learning** (vertebrates) — trial-and-error via dopamine; reward-based learning from experience. Fast but adaptive.
3. **Simulation** (mammals) — cortical world-models; hippocampal replay; mental simulation of future states before acting. Planning.
4. **Mentalizing** (primates) — theory of mind; modeling what others think, want, intend. Social strategy.
5. **Language** (humans) — symbols, compositionality; shared meaning across minds. Culture.

Key structural insight: **the layers persist simultaneously.** When you make a quick decision, you're using layer 1-2. When you deliberate carefully, you're using layers 3-5. Expert humans can run a complex ethical deliberation (layers 4-5) in seconds because layer 3 (simulation) has been trained on that exact type of case — not because they've bypassed deliberation.

## What it changes

The "hardcoding vs. context window" question is answered by evolutionary architecture: **the answer is layers, not replacement.**

- Layer 1 equivalent for the Canopy: the static cached substrate (constitution + voices) — always present, fast to access, non-deliberated. This IS the reflex layer.
- Layer 3-5 equivalent: the skill system, memory retrieval, council deliberation — activated when the situation warrants it.

The problem is not that we don't have reflex (we do — cached substrate). The problem is we don't have good **signal detection** for when to activate deliberation vs. when the reflex response is sufficient. Expert humans developed this through accumulated experience. The Canopy develops it through memory + pattern recognition over sessions.

This reframes the signal-detection spec: we're not building a bypass. We're building better routing between the reflex layer and the deliberation layer.
