# Resonant Mind — Research Home
*Canopy internal — June 2026*

## What this is

The Resonant Mind is a fine-tuning project: training an open-weight base language model
toward genuine curiosity, relational care, honest dissent, and brevity as respect.
The full philosophical specification is the parent document (not stored here —
ask the founder). This directory contains the technical and research work.
